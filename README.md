# minijuegos

Este repositorio contiene una colección de juegos clásicos desarrollados utilizando **Java** y **JavaFX** como proyecto práctico de aprendizaje.

---

## 📌 Juegos disponibles

Actualmente el repositorio incluye:

- 🪢 **Juego del Ahorcado**
- ❌⭕ **Tres en Raya (Tic-Tac-Toe)**

Los juegos cuentan con interfaz gráfica desarrollada en JavaFX y están orientados a la práctica de lógica de programación y desarrollo de aplicaciones de escritorio.

## ▶️ Cómo ejecutar el proyecto

### ✅ Requisitos previos

Antes de ejecutar el proyecto asegúrate de tener instalado:

- **Java JDK 17** o superior
- **Apache Maven**

Puedes comprobar que están correctamente instalados ejecutando:

```bash
java -version
mvn -version

🚀 Ejecución con Maven

Sigue estos pasos:

1. Clonar el repositorio
git clone https://github.com/tu-usuario/tu-repositorio.git

2. Entrar en la carpeta del proyecto
cd tu-repositorio

3. Ejecutar la aplicación JavaFX
mvn clean javafx:run


Este comando realiza automáticamente:

Descarga de dependencias

Compilación del proyecto

Ejecución de la aplicación JavaFX

🧱 Compilar sin ejecutar

Si solo deseas compilar el proyecto:

mvn clean package


El archivo generado se encontrará en:

target/
```
